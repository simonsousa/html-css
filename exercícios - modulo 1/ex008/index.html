Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>(não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong>(semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>(não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

A tag <b>, e a tag <i>, são tags da época do HTML4, ou seja, para representar formatos, as CSS já são responsáveis, tornando inviável, utilizar tags não semânticas para HTML5. Portanto, acostume-se a utilizar as tags <strong> para Negrito e <em> para itálico
